🍽️ Restaurant System (Java Console App)

This project is a console-based restaurant management system developed in Java.
It simulates an interactive experience between a customer and an electronic restaurant,
allowing the user to browse the menu, add meals, customize orders, and confirm them.

🧩 Project Structure

The program is divided into multiple stages, each represented by a separate Java file:

Stage	File Name	Description
1️⃣	AccueilRestaurant.java	Displays the welcome screen and main menu options.
2️⃣	MenuRestaurant.java	Shows available meals and allows editing or deleting orders.
3️⃣	QuantitySelection.java	Manages quantity selection, extras (like mayonnaise, harissa, cheese), and cart summary.
⚙️ Features

View food menu

Choose meal and quantity

Add optional extras

Edit or delete orders

Confirm the order and view total price

Display shopping cart and final receipt

🚀 How to Run

Make sure you have Java JDK installed.

Open the project folder in VS Code or any IDE.

Compile all files:

javac *.java


Run the main class:

java AccueilRestaurant
