# Restaurant Management System

## Description
A Java application for managing restaurant operations such as menus, orders, and basic workflow.

## Features
- Manage menu items
- Manage orders
- Add, update, and delete records
- Display restaurant data
- Simple user interface

## Technologies Used
- Java
- Git
- GitHub

## What I Learned
- Object-Oriented Programming
- CRUD operations
- Application structure
- Managing code with Git and GitHub
